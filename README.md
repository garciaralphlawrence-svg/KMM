# KMM Travel Co.

Static travel website for KMM Travel Co. This project is ready for GitHub Pages.

## Publish as a GitHub Pages project site

1. Create a new GitHub repository, for example `kmm`.
2. In this folder, run:

```bash
git init
git add .
git commit -m "Prepare KMM Travel Co. site for GitHub Pages"
git branch -M main
git remote add origin https://github.com/<your-username>/kmm.git
git push -u origin main
```

3. In GitHub, open the repository settings and enable GitHub Pages from the `main` branch and `/root`.

Your public link will be:

```text
https://<your-username>.github.io/kmm/
```
